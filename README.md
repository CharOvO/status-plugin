# STATUS-PLUGIN

- 一个适用于 [Yunzai 系列机器人框架](https://github.com/yhArcadia/Yunzai-Bot-plugins-index) 的系统状态查询插件
- 有着更加精简，美观的 UI 和的风格，并且你可以自定义头图

## 安装插件

#### 1. 克隆仓库

```
git clone https://github.com/CharOvO/skr-status-plugin.git ./skr-status-plugin
```

> [!NOTE]
> 如果你的网络环境较差，无法连接到 Github，可以使用Gitee咕咕咕~

#### 2. 安装依赖

```
pnpm install --filter=skr-status-plugin
```

## 插件配置

> [!WARNING]
> 非常不建议手动修改配置文件，本插件快兼容 [Guoba-plugin](https://github.com/guoba-yunzai/guoba-plugin) 了，到时候请使用锅巴插件对配置项进行修改,咕咕咕~

## 功能列表

安装好后，请使用 `#状态` 或 `/status` 命令触发状态面板

- [x] CPU 占用,温度
- [x] 内存占用
- [x] 网络上下行速率
- [x] 磁盘占用,温度
- [x] 系统信息
- [ ] GPU 型号
- [ ] 插件数量
- [ ] 适配器信息
- [ ] 收发消息数量
- [ ] 好友 & 群数量
- [x] 运行时间
- [ ] [自定义展示](https://github.com/erzaozi/neko-status-plugin?tab=readme-ov-file#%E8%87%AA%E5%AE%9A%E4%B9%89%E5%B1%95%E7%A4%BA)
